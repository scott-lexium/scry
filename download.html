Installation guide

  1. Download the DMG for your Mac
    If you have an M1, M2, M3, or M4 Mac — download the arm64 build. For older Intel Macs, use x64. Not sure? Click the Apple menu → About This Mac.
  2. Open the DMG and drag to Applications
    Double-click the downloaded .dmg file, then drag scrcpy GUI to your Applications folder.
  3. Right-click → Open on first launch
    macOS will block the app the first time because it isn't signed with a paid Apple certificate. In Finder (not Launchpad), right-click the app and choose Open. Then click Open in the dialog. You only need to do this once.
  4. Allow it in System Settings if needed
    If you see "cannot be opened", go to System Settings → Privacy & Security, scroll down, and click Open Anyway next to the scrcpy GUI entry.
  1. Download the Windows installer
    Download scrcpy-gui-win-x64.exe (installer) or the portable .exe if you prefer not to install.
  2. Bypass Windows SmartScreen
    When Windows shows the blue "Windows protected your PC" screen, click More info (small link below the warning), then click Run anyway. The app is safe — SmartScreen warns about any unsigned executable.
  3. Complete the installer
    Follow the installer steps. By default it installs to C:\Program Files\scrcpy GUI and creates a Start Menu shortcut.
  4. Launch from the Start Menu
    Search for scrcpy GUI in the Start Menu, or use the desktop shortcut if you created one during installation.
  1. Download the AppImage or .deb
    Use the AppImage for any distro. Use the .deb for Debian, Ubuntu, Pop!_OS, and compatible systems.
  2. Make the AppImage executable
    Open a terminal in your Downloads folder and run:
    chmod +x scrcpy-gui-linux-x64.AppImage
  3. Run it
    Double-click the AppImage in your file manager, or run:
    ./scrcpy-gui-linux-x64.AppImage
  4. Or install the .deb package
    sudo dpkg -i scrcpy-gui-linux-x64.deb
